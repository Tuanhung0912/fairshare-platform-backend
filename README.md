# FairShare - Backend API ⚙️ 💸

> **Core engine and API for the FairShare application.**

This repository contains the backend service for **FairShare**, a smart expense splitter and personal finance manager. It handles user authentication, data persistence, real-time synchronization, and the core debt simplification algorithms.

## 🎯 Planned Features

- **RESTful APIs:** Endpoints for user management, personal wallets, and group rooms.
- **Authentication & Authorization:** Secure endpoints using Spring Security and JWT.
- **Real-time Notifications:** WebSocket integration to notify users of new expenses or settled debts instantly.
- **Database Management:** Efficient schema design to track complex transactional histories.

## 🛠️ Tech Stack (Proposed)

- **Framework:** Java, Spring Boot
- **Database:** PostgreSQL / MySQL (with Spring Data JPA)
- **Security:** Spring Security, JWT
- **Real-time:** WebSockets
- **Deployment:** Docker (Future integration)
